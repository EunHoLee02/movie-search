# MUI Movie Finder

React, MUI, TMDB API를 사용한 영화 검색 앱 완성 예시입니다.

## 기능

- 현재 상영작 자동 표시
- 영화 제목 검색
- 로딩 / 에러 / 결과 없음 상태 처리
- 영화 카드 목록
- 영화 상세 Dialog
- MUI 기반 UI

## 실행 방법

```bash
npm install
npm run dev
```

## 환경 변수

프로젝트 루트에 `.env` 파일을 만들고 아래 값을 넣어야 합니다.

```env
VITE_TMDB_ACCESS_TOKEN=your_tmdb_access_token
```
